# 测试用户登录注册功能

**测试目标**: 验证用户登录和注册功能是否正常

**前置条件**:

- 测试环境：{{url}}
- 测试账号：{{account}}
- 密码：{{password}}
- 如果登录失败，请先注册一个账号

**已知限制**:

- 页面存在两套表单（登录表单+注册表单），需确保填写正确的表单并点击正确的按钮
- 表单使用 GET 方法提交，凭证会暴露在 URL 中（已知问题，不影响测试）

**测试步骤**:

1. **登录**

- 打开浏览器，访问 {{url}}/login
- **重要**：页面有两套表单，请确保填写第一套登录表单（上半部分）
- 输入账号和密码
- 点击第一套表单的登录按钮（"Sign in"按钮）
- 验证登录成功，跳转到主页（URL 应为 {{url}}/）

2. **注册**（如果登录失败）

- 如果登录失败（提示账号不存在），点击"Sign up"注册按钮
- 页面跳转到注册页（标题为"Create an account"）
- 输入邮箱、昵称和密码（邮箱使用 {{backup_account}}，昵称 test1）
- 点击"Continue"按钮
- 验证注册成功，跳转到登录页面（URL 应为 {{url}}/login）

3. **重新登录**

- 输入刚注册的账号和密码
- 填写第一套登录表单
- 点击登录按钮
- 验证登录成功，跳转到主页
- 确认页面显示"Welcome to RAGFlow"和 Dataset、Chat 等功能区域

**预期结果**:

- 账号不存在时，注册流程正常
- 注册成功后可正常登录
- 登录成功后 URL 变为 {{url}}/，显示主页内容
- 主页包含 Dataset、Chat、Search、Agent、Memory 等功能区域
