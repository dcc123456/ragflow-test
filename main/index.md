## 测试执行

### 执行流程

#### 串行执行
```
for each 任务 in 选中任务列表:
    1. 等待上一个任务完成（如有）
    2. 执行任务前检查
    3. 使用 web-test-agent 执行任务
    4. 截图保存
    5. 记录日志
    6. 通知进度/完成
    7. 执行环境清理（如需要）
    8. 汇总结果
```

#### 并行执行
```
for each 任务 in 选中任务列表:
    启动独立线程执行:
        1. 使用独立的浏览器实例
        2. 执行任务
        3. 截图保存
        4. 记录日志
等待所有线程完成
汇总结果
```

### 资源限制配置

| 资源 | 限制值 | 超过处理 |
|------|--------|----------|
| 单步超时 | 30秒 | 重试或标记失败 |
| 任务超时 | 5分钟 | 强制终止并标记失败 |
| 重试次数 | 3次 | 超过则标记失败 |
| 并行数 | 3个 | 超过则排队等待 |

### 错误恢复

| 错误类型 | 处理方式 | 重试策略 |
|----------|----------|----------|
| 页面元素未找到 | 标记可疑元素，重试 | 重试3次，间隔2秒 |
| 网络超时 | 检查网络，重试 | 重试2次，间隔5秒 |
| 浏览器崩溃 | 重启浏览器实例 | 重试1次，仍失败终止 |
| API响应错误 | 记录错误码，重试 | 重试2次，间隔3秒 |
| 断言失败 | 截取当前状态 | 不重试，直接标记失败 |

### 环境清理

| 任务类型 | 清理内容 | 清理时机 |
|----------|----------|----------|
| add-llm | 删除添加的模型提供商 | 测试完成后 |
| data-upload | 清理上传的文件 | 测试完成后 |
| 通用 | 登出当前用户 | 测试完成后 |

## 测试用例编写规范

### 文件命名
- 格式：`{功能名}-test.md`
- 示例：`billing-test.md`、`login-test.md`

### 文件结构

```markdown
## 测试任务名称

**测试目标**: 简要说明测试目的

**前置条件**:
- 环境要求
- 账号要求
- 数据要求

**测试数据**:
| 数据类型 | 值 | 说明 |
|----------|-----|------|
| URL | http://xxx | 测试地址 |
| 账号 | xxx | 用户名/密码 |
| 文件 | path/to/file | 上传文件路径 |

**执行步骤**:
1. 步骤描述
2. 步骤描述
   - 预期结果
3. 步骤描述
   - 预期结果

**环境清理**:
- 清理内容1
- 清理内容2
```

### 编写要求

| 要求 | 说明 |
|------|------|
| 步骤明确 | 每步只做一件事，避免复合步骤 |
| 结果可判 | 每步有明确的预期结果，可自动判定 |
| 独立性强 | 步骤间无隐式依赖，可单独执行 |
| 参数化 | URL/账号等用变量占位，便于替换 |
| 时效性 | 定期更新账号/URL等过期信息 |

### 步骤描述模板

```
1. **操作描述**
   - 操作：点击/输入/选择/上传...
   - 目标：元素定位器或描述
   - 预期：页面变化或结果
```

## 回归测试

当项目代码变更后，需要执行回归测试：

1. **确定变更范围**：了解本次变更影响的功能
2. **选择相关用例**：只执行受影响的测试任务
3. **执行回归测试**：按流程执行选中的测试
4. **对比历史结果**：与上次测试结果对比，确认无退化
5. **记录回归结果**：保存回归测试报告

**回归测试触发条件**：
- 代码提交后
- 新功能合并后
- Bug修复后
- 定期巡检（建议每周一次）

## 安全注意事项

| 注意事项 | 说明 |
|----------|------|
| **敏感信息** | 禁止在测试用例中硬编码真实密码/API Key |
| **日志脱敏** | 测试日志中的敏感信息需要打码 |
| **截图保护** | 截图中的敏感信息需要模糊处理 |
| **环境隔离** | 测试环境与生产环境严格分离 |
| **数据清理** | 测试完成后及时清理敏感数据 |

### 敏感信息处理

```javascript
// 脱敏函数
function maskSensitive(str) {
  if (!str) return '';
  if (str.length <= 4) return '****';
  return str.slice(0, 2) + '****' + str.slice(-2);
}

// 使用
console.log(maskSensitive('sk-abc123def456'));
// 输出: sk****56
```

## 常见问题处理

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| MCP连接失败 | 服务未启动 | 重启 MCP 服务 |
| 页面加载慢 | 网络问题 | 检查网络，增加超时时间 |
| 元素定位失败 | 页面结构变化 | 更新定位器，通知开发 |
| 账号被踢出 | 会话过期 | 重新登录，更新用例 |
| 截图模糊 | 页面还在加载 | 增加等待时间 |
| 浏览器无响应 | 内存不足 | 重启浏览器，增加等待 |
| Cookie失效 | 会话过期 | 清除缓存，重新登录 |

## 测试指标

### 统计指标

| 指标 | 计算方式 | 目标值 |
|------|----------|--------|
| 通过率 | 通过任务/总任务×100% | ≥90% |

- 每轮测试的通过率变化
- 各任务平均耗时变化
- 失败任务的分布
- 用例完善数量
